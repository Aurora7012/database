脚本主体：校园网自动登录脚本
作用：自动登陆到校园网
脚本配置器：校园网登录脚本 配置器
作用：配置登陆脚本所使用的账号、密码；设置是否开机自启。
配置文件所在位置：D:\AuroraApp\Login（第一次启动配置器后才会生成）

注意：使用配置器中的开机自启功能需要管理员权限！

*********************************************************

脚本主体会连接wifi，连接成功后会弹出登录框。
此时已经完成登录(尽管显示登陆窗体)，直接关闭浏览器即可。
如果不想手动关闭浏览器窗口，请按照如下步骤禁用自动检测网络弹窗认证：
1、Win10用户请先以管理员身份运行 [ Win10组策略安装.cmd ]
2、使用快捷键 Win + R
3、输入 gpedit.msc 并确定(回车即可)
4、按照 [禁用win10自动检测网络弹窗认证方法图示.png]提到的方法 启用 即可
